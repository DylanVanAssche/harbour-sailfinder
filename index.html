WHAT'S SAILFINDER?

Sailfinder is an unofficial Tinder client for Jolla's Sailfish OS.

Our goal?
  • Provide native access to Tinder for Sailfish OS users.
  • Show Tinder that Sailfish OS is ready for an official native application.
Features
  • Discover people in your area, like, dislike or superlike them.
  • View people their profile
  • Chat with your matches
  • GIF support
  • Edit your profile, upload new pictures, ...
  • Native notifications
  • Available in several languages, for more information click here
  • Modern C++ backend with a beautiful Sailfish OS UI
  • Open source, available on Github
  • Many more!
Support

Sailfinder is completely developed in spare time, if you want to support the development further you can buy us a coffee.

Jolla Store

The official Jolla Store where all stable releases are available.

Openrepos.net

Experimental features are tested on Openrepos.net before they are available in the Jolla Store.
Download the latest Sailfinder from Openrepos.net or install Sailfinder using the Storeman client.
